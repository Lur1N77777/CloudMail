# Roadmap

This roadmap tracks useful future improvements. It is not a release commitment.

## Potential improvements

- Attachment support for compatible mailbox backends.
- Webhook settings for supported backends.
- More complete local attachment downloads.
- Push notifications.
- Biometric unlock.
- QR-based account export and import.
- More complete internationalization.

## Maintenance

- Keep API behavior aligned with the supported mailbox backend.
- Keep APK distribution in GitHub Releases instead of committing binary files.
- Keep documentation updated when setup, build, or configuration changes.
