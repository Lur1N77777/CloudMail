# Notices and attribution

CloudMail is a mobile admin client designed for mailbox systems compatible with:

- Project: cloudflare_temp_email
- Repository: https://github.com/dreamhunter2333/cloudflare_temp_email
- Author: dreamhunter2333 and contributors

The mailbox backend/API behavior targeted by CloudMail comes from the upstream Cloudflare temporary email project. CloudMail adds a React Native / Expo mobile administrator experience, local grouping, mobile mail viewing, and release packaging around that ecosystem.

If you use CloudMail with the upstream mailbox system, follow the upstream project's license and deployment documentation.
